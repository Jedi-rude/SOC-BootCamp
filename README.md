###  BootCamp Objectives
This bootcamp help to understand what are the basic operation for the Security Operation Center Monitoring (SOC) operations and basic system engineering concepts.

- NTP

- WEB
- SSH
- Proxy

Monitoring and Security Services
- RADIUS
- Rsyslog
- Wazuh
- SSH Guard
